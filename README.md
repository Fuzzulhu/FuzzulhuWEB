# FuzzulhuWEB

Repositorio oficial para la página web de la banda de Stoner Blues Doom **Fuzzulhu** (Concepción, Chile).

## Identidad Visual
- **Colores:** Negro profundo, gris oscuro, y tonos morado/violeta como acento principal. Textos blancos para contraste.
- **Estética:** Texturas de papel rasgado, imágenes con alto contraste, filtros oscuros, imaginería lovecraftiana/cósmica.
- **Lema:** "Stoner Blues Doom"

## Estructura de la Web (One-Page / Multi-Page)
Toda la información proviene del Dossier oficial de la banda.

1. **Inicio (Home):** Imagen principal con túnicas, logo y CTA sobre el lanzamiento del primer EP.
2. **Nosotros:** "Fuzzulhu es una banda de Stoner Blues Doom que emerge en el año 2019 en Concepción, Chile. Nace de la necesidad de expresar ideas musicales plagada de tintes oscuros... abordando temas de lo urbano y lo rural, leyendas, mitos... imaginaria de H.P. Lovecraft o Poe."
3. **Miembros:** Felipe Viecho López (Guitarra y voz), Pato Andrés Soto (Guitarra y coros), Pablo Sandoval (Bajo), Rodrigo Jofré (Batería).
4. **En Vivo (Tour):** Fechas destacadas como Sepulchral rites, Festival Puerto vivo, Culto al pulpo III, etc.
